## 关于本作

基于日本民间传说「佐子」（サチコ）改编的群像式恐怖文字推理游戏。

玩家将从五个视角——转学生、施暴者、旁观者、调查者、亡灵——分别进入同一座被雪覆盖的小镇，拼凑出一个女孩从死亡到诅咒的完整真相。每一条路线都是碎片，只有全部走完，才能听见那首歌的最后一个音符。

> **警告：** 本作包含心理恐怖、暗示性暴力、校园霸凌、非直白的死亡描写。不建议 18 岁以下玩家体验。请佩戴耳机，在暗处独自游玩。

## 特性

| | |
|---|---|
| **五条角色路线** | 雾岛朔（转学生）· 黑川诚（施暴者）· 浅野由纪（旁观者）· 九条凛（调查记者）· 佐子（解锁条件：完成任意两条非佐子路线） |
| **44 个结局** | 每条路线 7-13 个分支结局，含生存、死亡、救赎、诅咒、传唱者等多种类型 |
| **三层模式选择** | 角色 × 强度（叙事/诅咒/噩梦）× 玩法（故事/推理/生存），共 45 种组合 |
| **理智与诅咒系统** | 双状态值实时影响画面、音效、闪现频率与结局走向 |
| **线索收集** | 跨路线共享的线索与真相档案，New Game+ 中可触发隐藏内容 |
| **随机事件** | 每次游玩基于种子生成不同的环境事件与闪现 |
| **程序化音效** | Web Audio API 实时合成：风雪、列车、心跳、钢琴、童谣——零音频文件 |
| **日式恐怖视觉** | 噪点、扫描线、暗角、雪花、文字故障、审查黑条、SVG 闪现——纯 CSS 实现 |
| **打字机引擎** | 可控速度、跳过、暂停，配合文字特效（震颤/血渗/镜像/消音） |
| **存档系统** | localStorage 自动保存进度、解锁内容、设置项 |

## 快速开始

### 方式一：直接打开

下载 `index.html`，双击用浏览器打开即可。无需安装、无需联网。

### 方式二：本地服务器

```bash
# 在项目目录下任选一种
python -m http.server 8000
# 或
npx serve
```

然后访问 `http://localhost:8000`。

> **推荐浏览器：** Chrome / Firefox / Edge / Safari 最新两版

## 玩法说明

### 角色选择

| 角色 | 身份 | 视角特点 |
|------|------|----------|
| **雾岛朔** | 转学生 | 初来乍到，以陌生人的眼睛看见异常 |
| **黑川诚** | 施暴者 | 带着罪恶感生活，从加害者一侧直面报应 |
| **浅野由纪** | 旁观者 | 当年沉默的目击者，在愧疚与自保之间挣扎 |
| **九条凛** | 调查记者 | 外来者，用理性与证据试图揭开三年前的真相 |
| **佐子** | 亡灵 | 解锁后可玩。从死者的视角，重新经历自己的死亡 |

### 强度选择

| 强度 | 说明 |
|------|------|
| **叙事** | 无强制限时，适合沉浸阅读与剧情体验 |
| **诅咒** | 部分关键选择限时，闪现频率随诅咒值升高 |
| **噩梦** | 多数选择限时，心跳音效常驻，闪现极频繁 |

### 玩法模式

| 模式 | 说明 |
|------|------|
| **故事** | 以叙事为主，额外场景与对话，适合首次游玩 |
| **推理** | 强调线索收集与逻辑判断，部分选项需要持有特定线索才解锁 |
| **生存** | 理智值持续下降，错误选择直接致死，适合挑战极限 |

## 解锁系统

游戏中的所有解锁内容会自动保存到浏览器本地存储，包括：

- **结局** — 所有达成过的结局
- **死亡** — 所有经历过的死亡方式
- **遭遇** — 特殊随机事件
- **童谣变体** — 不同路线中听到的佐子之歌变体
- **真相档案** — 隐藏的世界观文档

> 完成**任意两条非佐子路线**后，主菜单将出现第五个角色——**佐子**。

## 技术细节

```
文件结构
├── index.html          ← 全部游戏内容（326KB / 6770行）
├── .gitignore
└── README.md
```

- **零依赖**：无外部 CSS / JS / 字体 / 音频 / 图片文件
- **零网络请求**：完全离线运行
- **单文件部署**：复制一个 `index.html` 即可分发
- **技术栈**：原生 HTML + CSS + JavaScript（无框架、无构建工具）
- **音频**：Web Audio API 实时合成（OscillatorNode / BufferSource / 噪声生成）
- **视觉**：纯 CSS（噪点用 SVG data URI，雪花用 Canvas，闪现用 SVG overlay）

## 佐子传说

> 佐子（サチコ）是日本校园怪谈中的经典形象。
>
> 传说中，她是一名被同学霸凌、推入暗沟致死的少女。她的双腿折断，在雪中冻死。从此以后，每逢雪夜，她的歌声会在校园里响起。听到歌声的人，会在雪中看见一个没有腿的女孩——她不追你，只是站在那里，唱着那首没有名字的歌。
>
> 但如果你听完了整首歌……
>
> 她会记住你的名字。
